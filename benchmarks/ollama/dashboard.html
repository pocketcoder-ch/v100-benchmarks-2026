Грабли и аномалии

Методология: 1 inference на каждую (model, level), N=1 без повторов. num_predict=300, temperature=0.0, num_ctx=22000.
Корпус prompts: реальные транскрипции звонков колл-центра, 4 уровня — short ~3K, small ~5K, medium ~10K, large ~14K tokens.
Eval prompt: диагностика боли клиента (~250 tok). Цветовая шкала decode_tps: ≥100 / 60-99 / 40-59 / 25-39 / 15-24 / 5-14 / <5.
Env Ollama: NUM_PARALLEL=1, FLASH_ATTENTION=1, KV_CACHE_TYPE=q8_0 (главное для 70B+ TP=2).
Pod topology: pod1=NV1, pod2=SYS (PCIe), pod3/pod4/pod5=NV2 (NVLink). 70B+ автоматом TP=2.
Decode_tps: output_tokens / (total_wall_ms − ttft_ms). Prefill_tps: prompt_tokens / ttft_sec.
Слабость: N=1 даёт ±5-10% jitter. Для статьи показываем порядок величин; для прода нужно N≥3.
