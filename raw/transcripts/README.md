# Transcripts

Four real BVM call-center transcripts in Russian, used as input context for the LLM benchmark at four length levels: `short.txt` (~3K tokens), `small.txt` (~5K), `medium.txt` (~10K), `large.txt` (~14K).

Each file is a transcribed sales/support phone call (Спикер 0 = manager, Спикер 1 = client) plus segment metadata. The transcripts have been anonymized — names, phone numbers, and any client identifiers stripped or replaced — and are released under CC-BY-4.0 alongside this repo. Use them as a realistic Russian-language context payload; they are not intended as training data.
